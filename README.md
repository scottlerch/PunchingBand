# Punching Band
Fun boxing exercise app for your Microsoft Band 1 or 2, sadly never to be 3 :(
[![Punching Band Demo](https://j.gifs.com/Vmr6QX.gif)](https://youtu.be/GPvN8QixsKE)  
https://youtu.be/GPvN8QixsKE

# Features
* Detects punch strength using accelerometers
* Punch recognition (jab, hook and upper cut) using accelerometers/gyroscopes data with Accord.NET back-propogation neural network
* Cross platform with Windows, iOS and Android via Xamarin
* Winner of the Microsoft 2015 OneWeek Hackathon's Mission Impossible II award
